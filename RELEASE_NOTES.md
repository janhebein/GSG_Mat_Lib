# Release Notes - GSG Mat Lib v4

Release date: 2026-03-04

## Highlights
- Improved drag/drop reliability for panel-to-network workflows.
- Direct texture/HDRI drop-to-parameter behavior for compatible nodes.
- Reduced noisy drop-path logging and cleaner interaction feedback.
- Better thumbnail handling with on-demand cache generation for large/heavy formats.
- Correct per-texture preview rendering in material detail views.
- Octane builder workflow improvements, including shared transform wiring and displacement defaults.

## Core Features
- Browse libraries by **Materials**, **Textures**, and **HDRIs**.
- Drag and drop materials into network editors.
- Drag and drop textures/HDRIs onto compatible node file parameters.
- Context actions for copying file/material paths.
- Cached thumbnail previews for faster browsing.
